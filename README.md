# 霓虹战机 (Neon Strike)

微信小游戏 - 竖版弹幕射击游戏（类似雷电/1942）

## 项目结构

```
neon-strike/
├── game.js                  # 入口文件（加载 main.js）
├── game.json                # 游戏配置
├── app.json                 # 应用配置
├── project.config.json      # 微信项目配置
├── images/                  # 图片资源（当前为空）
├── audio/                   # 音频资源（当前为空）
└── js/
    ├── main.js              # 全部游戏逻辑（~1800行）
    ├── render.js            # 渲染模块（桩文件）
    ├── databus.js           # 数据总线（桩文件）
    ├── libs/
    │   └── tinyemitter.js   # 事件发射器（未使用）
    ├── base/
    │   ├── sprite.js        # 精灵基类（桩文件）
    │   ├── animation.js     # 动画类（桩文件）
    │   └── pool.js          # 对象池（桩文件）
    ├── player/
    │   ├── index.js         # 玩家类（桩文件）
    │   └── bullet.js        # 子弹类（桩文件）
    ├── npc/
    │   └── enemy.js         # 敌人类（桩文件）
    └── runtime/
        ├── background.js    # 背景类（桩文件）
        ├── gameinfo.js      # 游戏信息（桩文件）
        └── music.js         # 音乐类（桩文件）
```

> **注意**：所有游戏逻辑集中在 `js/main.js`，其他 JS 文件均为桩文件，尚未实现。

## 游戏系统

### 核心类（js/main.js）

| 类名 | 行号 | 功能 |
|------|------|------|
| ObjectPool | 46-96 | 对象池，复用子弹/敌人/道具/粒子对象，减少 GC |
| SpatialGrid | 102-171 | 空间网格，碰撞检测优化 O(n²) → O(n) |
| Entity | 177-215 | 实体基类，位置/速度/碰撞半径 |
| Player | 221-383 | 玩家飞机，HP/火力等级/无敌帧/护盾 |
| Bullet | 389-429 | 子弹（玩家/敌人共用） |
| Laser | 435-563 | 激光武器，含闪电链效果 |
| Enemy | 569-825 | 敌机（4种类型 + Boss三阶段） |
| Item | 831-952 | 道具掉落（火力/炸弹/护盾/分数） |
| Particle | 958-1010 | 粒子特效（爆炸/火花/擦弹） |
| Starfield | 1016-1072 | 星空滚动背景 |

### 游戏状态

- **menu** — 开始界面
- **playing** — 游戏进行中
- **gameover** — 游戏结束，显示分数

### 玩家系统

| 属性 | 数值 |
|------|------|
| 生命值 | 3 |
| 初始炸弹 | 1 |
| 受伤无敌时间 | 2秒 |
| 判定点半径 | 6px |
| 擦弹半径 | 25px |
| 射击间隔 | 100ms |

**火力等级：**

| 等级 | 武器 |
|------|------|
| Lv.1 | 2发直射子弹 |
| Lv.2 | 5发扇形弹（±20°） |
| Lv.3 | 10发扇形弹（±30°，双排） |
| Lv.4 | 自动追踪激光 |

### 敌机系统

| 类型 | HP | 分数 | 速度 | 移动方式 | 弹幕 |
|------|----|------|------|----------|------|
| Scout（侦察机） | 1 | 100 | 150 | 直线 | 无 |
| Fighter（战斗机） | 3 | 300 | 180 | 蛇形 | 3路扇形（±30°） |
| Elite（精英机） | 15 | 1000 | 100-150 | 追踪玩家 | 追踪弹 |
| Boss（Boss） | 500 | 5000 | 30-80 | 8字形 | 三阶段弹幕 |

**Boss 三阶段弹幕：**
- 阶段1（HP > 50%）：12路旋转圆弹
- 阶段2（HP 25-50%）：8路集中向下
- 阶段3（HP < 25%）：16路密集旋转

### 道具系统

| 道具 | 效果 | 掉落率 |
|------|------|--------|
| P（火力） | 火力等级+1（最高4级） | 普通敌机10% / 精英50% / Boss 100% |
| B（炸弹） | 炸弹+1（最多3个） | 同上 |
| S（护盾） | 抵挡一次伤害 | 同上 |
| 2x（分数） | 分数翻倍10秒 | 同上 |

道具在靠近玩家150px时会自动吸附。

### 计分系统

- 基础分数：Scout 100 / Fighter 300 / Elite 1000 / Boss 5000
- 连击加成：每10连击 +10%
- 擦弹奖励：50分/次
- 道具翻倍：2x道具生效期间分数翻倍

### 难度递增

- 刷敌间隔：1500ms → 500ms（随波次递减）
- 敌机类型逐步解锁：Scout → Fighter → Elite
- Boss 在 1000 分时出现
- 每波敌机数量随波次增加

### 碰撞检测

使用 SpatialGrid（网格大小80px）优化：
- 玩家子弹 vs 敌机：圆-圆碰撞
- 激光 vs 敌机：线-圆碰撞
- 敌机子弹 vs 玩家：双半径（擦弹25px / 判定6px）
- 敌机 vs 玩家：机体碰撞
- 道具 vs 玩家：拾取半径30px

## 渲染

- **技术**：Canvas 2D API，纯程序绘制（无图片资源）
- **绘制顺序**：背景 → 道具 → 敌机 → 激光 → 子弹 → 粒子 → 玩家 → UI → 闪屏
- **特效**：引擎火焰闪烁、护盾脉冲、爆炸粒子、闪电抖动、全屏闪白

## 微信 API 使用

- `wx.createCanvas()` — 创建画布
- `wx.getSystemInfoSync()` — 获取屏幕尺寸
- `wx.onTouchStart/Move/End()` — 触控输入
- `wx.getStorageSync()` / `wx.setStorageSync()` — 本地存储最高分

## 操作说明

- 手指滑动控制飞机移动
- 自动射击，无需手动操作
- 点击右上角区域（屏幕20%宽×30%高）使用炸弹

## 运行方式

1. 使用微信开发者工具打开项目目录
2. 修改 `project.config.json` 中的 `appid`
3. 点击编译运行
